Rmh-homebasecivi

Instructor and Developer Resources

Download this project as a .zip file Download this project as a tar.gz file

Instructors or developers who are interested in adapting RMH-Homeroomcivi for a different non-profit application may freely download and use any of the following resources as a starting point:

  1. Design document
  2. Code base
  3. "Sandbox" database
  4. Course syllabus
  5. Weekly assignments
  6. Final class presentation
  7. Student evaluations

These resources are all licensed under a Creative Commons Attribution-ShareAlike 3.0 Unported License.

Detailed guidance on adapting these materials to a new course is available in our book "Open Source Software Development," which will soon appear as a Kindle book.

For on-line access to a "sandbox" version of this software, log in at rmhhomebasecivi.myopensoftware.org with Username Jane2078293469 and Password tryme.
